# CleanStore

### Приложение для управления каталогом товаров с авторизацией пользователей и ролями. 
Реализовано на WPF с использованием Entity Framework.

## *Функции*
* Авторизация пользователей
* Разделение ролей (гость / администратор)
* Просмотр списка продукции
* Добавление / удаление товаров
* Редактирование товаров
* Работа с базой данных через Entity Framework

## *Технологии*
* C#
* WPF
* Entity Framework
* SQL Server

## *Структура проекта*
* Windows — окна приложения (WPF)
  * LoginWindow: 
    1.	Поля ввода для логина и пароля
    2.	Кнопка «Гость» только для просмотра
    3.	Кнопка «Войти» для входа пользователя с возможностью управления товарами
  * MainWindow
    1.	Кнопка «Выйти» для разлогина и перехода к новой авторизации
    2.	Кнопка «Товары» для просмотра таблицы продукции
  * ProductWindow
    1.	Кнопка «Назад» для выхода в главное меню
    2.	Поле ввода для поиска товара
    3.	Выводящиеся списки для фильтрации по поставщику и сортировки
    4.	Кнопки «Добавить», «Удалить», «Редактировать» для перехода к соответствующему действию
  * ProductEditWindow
    1.	Поля ввода и списки вывода с информацией о добавляемом или редактируемом товаре
    2.	Кнопка «Сохранить» для добавления или подтверждения редактирования товара в БД
    3.	Кнопка «Отмена» для закрытия страницы редактирования
